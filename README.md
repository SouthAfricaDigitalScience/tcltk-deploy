# tcltk-deploy

Build, test and deploy scripts for [Tcl/Tk](https://tcl.tk/)

# Dependencies

This project has no dependencies

# Versions

Versions built are:

# Configuration

The builds are configured with :

```
--enable-64bit \
--enable-shared \
--enable-threads \

```
